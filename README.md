# CVTurbo — Landing Page de Alta Conversão
<img width="1335" height="643" alt="image" src="https://github.com/user-attachments/assets/0d0afea3-99d3-47f3-a3e2-0a450260c4d0" />

O CVTurbo é uma plataforma focada em resolver um dos maiores gargalos de quem está à procura de emprego: o tempo gasto na personalização de currículos para cada vaga. Esta landing page foi desenhada com foco em Copywriting Persuasivo, Design Futurista e Conversão.

## 🚀 O Conceito

A ideia central é vender "Tempo" e "Sucesso em Entrevistas", não apenas um simples gerador de ficheiros PDF.

## Estratégia de Marca (Branding)

**Nome:** CVTurbo. O nome remete a velocidade, potência e aceleração de resultados na carreira.

**Paleta de Cores:** Fundo "Dark Mode" (#0D0E11) com detalhes em "Azul Elétrico/Ciano" (#00FFFF). Esta combinação transmite autoridade, tecnologia e inovação.

**Identidade Visual:** Uso de Glassmorphism (cartões com transparência e desfoque) e efeitos Neon para guiar o olhar do utilizador através de uma interface "high-tech" e moderna.

## Estrutura da Página (Copywriting)

**Hero Section:** Promessa clara. "Menos esforço, mais fluxo". O foco principal é na velocidade de entrega (1 minuto).

**Agitação do Problema (A Dor):** Identifica as frustrações comuns, como a perda de tempo em tarefas repetitivas e o descarte automático por sistemas de triagem (ATS).

**A Solução (O Método):** Apresenta o processo em 3 passos simples para reduzir a fricção cognitiva e mostrar simplicidade ao utilizador.

**Prova Social/Dados:** Uso de estatísticas impactantes (95% mais rápido, 3.5x mais respostas) para gerar credibilidade imediata.

**FAQ:** Antecipação de objeções comuns sobre segurança de dados e compatibilidade com recrutadores.

## 🛠️ Aspectos Técnicos

A página foi construída para ser extremamente leve e de carregamento instantâneo:

- **Tailwind CSS:** Para estilização responsiva sem necessidade de ficheiros CSS externos pesados.
- **Font Awesome:** Ícones vetoriais que garantem uma interface moderna e limpa.
- **Responsividade Total:** Layout adaptável para Mobile, Tablet e Desktop.

**Ajuste Milimétrico:** A linha do tempo no desktop foi calculada via CSS para alinhar perfeitamente com o centro dos ícones de cada passo (left: 50% com translateX(-50%)).

## 🤖 Prompt Utilizado (Engenharia de Prompt)

Para gerar esta landing page, o prompt estruturado foi o seguinte:

"Crie uma landing page de alta conversão para um SaaS chamado CVTurbo.
Objetivo: Captar leads para acesso antecipado (tester).
Público: Pessoas que buscam emprego mas estão exaustas de adaptar o currículo manualmente para cada vaga.

Requisitos de Design:

Estética 'Dark Tech' premium com fundo escuro e acentos em Ciano Neon.

Use Tailwind CSS e Font Awesome (tudo em um único arquivo HTML).

Implemente Glassmorphism nos cards.

Seção Hero impactante com título grande e CTA chamativo.

Seção de 'Dor' destacando perda de tempo e barreiras do ATS.

Seção de 'Como Funciona' com uma linha do tempo vertical (desktop) e centralizada. A linha deve passar exatamente por trás dos círculos numerados.

FAQ com efeito sanfona (accordion).

Botão CTA flutuante que aparece após o scroll.

Navbar fixa com efeito de desfoque (backdrop-blur).

Links de contato e redes sociais no footer.

Copywriting: Use tom profissional, porém inovador e empoderador. Foque na economia de tempo e na vantagem competitiva da IA."

## 📈 Publicação e Domínio

Para evitar conflitos com domínios já existentes, recomenda-se o uso de extensões modernas como .ai ou .io, que reforçam o posicionamento tecnológico da marca.

Como colocar no ar:

Guarda o código gerado anteriormente como index.html.

O ficheiro é autossuficiente (Single File) e pode ser publicado instantaneamente em serviços como Vercel, Netlify ou GitHub Pages apenas arrastando o ficheiro para o painel de upload da plataforma escolhida.
